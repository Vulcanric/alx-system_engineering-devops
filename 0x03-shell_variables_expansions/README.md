# Shell Scripting Tasks

This repository contains a collection of Bash scripts that perform various system administration and automation tasks.

## Scripts

- `alias.sh`: A script that creates an alias for the `ls` command to delete all files in the current directory.
- `path.sh`: A script that adds `/action` to the `PATH` environment variable.
- `countdirs.sh`: A script that counts the number of directories in the `PATH`.
- `listenv.sh`: A script that lists all environment variables.
- `listvars.sh`: A script that lists all local and environment variables, as well as functions.
- `newlocalvar.sh`: A script that creates a new local variable.
- `newglobalvar.sh`: A script that creates a new global variable.
- `addvars.sh`: A script that adds two environment variables together and prints the result.
- `powerdivide.sh`: A script that divides the value of one environment variable by another and prints the result.
- `powern.sh`: A script that raises one environment variable to the power of another and prints the result.
- `convert2to10.sh`: A script that converts a binary number to decimal.
- `numformat.sh`: A script that formats an environment variable as a number with two decimal places.
- `convert10to16.sh`: A script that converts a decimal number to hexadecimal.
- `rot13.sh`: A script that encodes and decodes text using the ROT13 encryption.
- `everyother.sh`: A script that prints every other line from a file.
- `addenvvars.sh`: A script that adds the values of two environment variables and prints the result.

Each script includes a brief description of its functionality and usage.
